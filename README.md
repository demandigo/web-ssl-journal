# web-ssl-journal

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式存储，不针对任何单个域名或具体网站，旨在提供可公开访问的文档与记录。

## 项目简介

`web-ssl-journal` 是一个以 HTML 页面为主的归档仓库。仓库中的每个页面均为独立文件，内容涉及与网络、安全相关的观察与记录。仓库本身不运行后端服务，所有页面可直接通过浏览器打开或通过 GitHub Pages 等静态托管服务访问。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 仓库首页 / 导航页
├── pages/             # 存放具体 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/            # 静态资源（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- **`index.html`**：仓库入口页面，提供页面列表或导航链接。
- **`pages/`**：所有独立 HTML 页面存放于此，按编号或主题命名。
- **`assets/`**：页面引用的公共资源文件，保持目录结构清晰。

## 页面归档说明

- 每个页面代表一个独立的归档条目，内容可能包含技术笔记、观察记录、学习笔记或其他与网络、安全相关的信息。
- 页面内容均以 HTML 格式撰写，不依赖外部数据库或动态服务。
- 归档内容不针对特定域名、网站或服务，仅作为通用记录与参考。
- 页面更新频率不固定，视归档材料积累情况而定。

## 维护说明
